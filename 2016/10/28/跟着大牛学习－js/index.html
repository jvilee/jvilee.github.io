记得一位前辈对我说过：虽然我自己是js开发的，但我一点也不想为js辩护，js语言本身很烂，一开始只是用来弹出 alert，后来由于前端开发迅速成长，只有js能够满足一些交互，造成js语言这栋烂尾楼越盖越高，也是不得已的事。现在 js语言也在不断完善。也就是后来的js的国际标准ECMAScript。这是我对js的最初印象。

插播一条我对开发语言的小看法，个人觉得不同语言都有其诞生的背景，语言是用来解决不同问题不同环境下的手段，在以后的职业生涯，我需要不断学习涉猎不同语言，以便遇到问题能够快速提出基本解决方案。

类型系统

JavaScript类型分为标准类型和对象类型,对象类型分为内置对象类型、普通对象模型、自定义对象模型

值类型与引用类型的区别
值类型储存在栈（stack）中储存变量的值，而引用类型在栈中保存的是所引用的内容存在堆（heap）中。引用类型储存的是地址而不是真实数值，对饮用类型的复制其实复制它的地址

Map与Set

map是es6规范新引入的数据类型－一组键值对，具有极快的查找速度(一个key对应一个value，多次存入，后面覆盖前面)

var m = new Map([[‘Michael’, 95], [‘Bob’, 75], [‘Tracy’, 85]]);
m.get(‘Michael’); // 95

Set与Map类似，它是一组key的集合，且不存入value，没有重复的key，重复的元素会被自动过滤掉

iterable

iterable也是es6标准新的类型，其中Array、Map和Set都属于iterable类型，同时该类型可以通过for…of来遍历,最好通过forEach遍历

浏览器

浏览器对象

  • window

window对象不但充当全局作用域，也可表示浏览器窗口，具有innerWidth和innerHeight两个属性(浏览器内部的宽度和高度) 对应的有outerWidth和outerHeight属性

  • navigator

表示浏览器的信息

  • screen

screen对象表示屏幕的信息

  • location

location对象表示当前页面的url信息

  • document(重要)

表示当前页面，html在浏览器中意DOM形式表示树形结构，document就是整个DOM树的根节点

  • history

history对象保存了浏览器的历史纪录,可以调用back()、forword() 后退、前进
